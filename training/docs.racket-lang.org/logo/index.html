1 Logo language

 #lang logo package: logo

1.1 Program grammar

 

‹program›

 ::= 

‹statement›*

A Logo program is a sequence of statements separated by whitespace. Comments begin with ; and continue until the end of the line.

1.1.1 Statements

 

‹statement›

 ::= 

‹procedure-definition›

 

  |  

‹procedure-call›

 

  |  

if   ‹test-expr›   ‹block›

 

  |  

repeat   ‹count-expr›   ‹block›

 

‹block›

 ::= 

[   ‹statement›*   ]

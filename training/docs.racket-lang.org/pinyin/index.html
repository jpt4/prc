1 API

procedure

(pinyin hanzi) → (listof string?)

  hanzi : char?
返回汉字对应的拼音.

Examples:
> (pinyin #\中)

'("zhōng" "zhòng")

> (pinyin #\国)

'("guó")

value

pinyin-hash-table : hash?

保存拼音数据的 Hash Table.

Examples:
> (hash-ref pinyin-hash-table #\中)

'("zhōng" "zhòng")

> (hash-ref pinyin-hash-table #\国)

'("guó")

Fast 1-D Haar Wavelet Transform

John Clements <clements@racket-lang.org>

 (require wavelet) package: wavelet-transform-haar-1d
This package provides functions to perform one-dimensional Haar wavelet transforms in linear time.

procedure

(fast-haar-transform array) → (Array Real)

  array : (Array Real)
Given a one-dimensional array of Reals whose size is a power of two, return the one-dimensional array of Reals of the same size that represents the result of the one-dimensional Haar transform.

procedure

(fast-haar-inverse-transform array) → (Array Real)

  array : (Array Real)
Given a one-dimensional array of Reals whose size is a power of two, return the inverse Haar transform. This function is the inverse of fast-haar-transform.

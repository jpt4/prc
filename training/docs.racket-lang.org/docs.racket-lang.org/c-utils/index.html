C Metaprogramming Utilities

by Dave Herman <dherman@ccs.neu.edu>

This package provides utilities for manipulating C programs, including a library for extracting binary layout information from header files for use with the foreign library—see The Racket Foreign Interface.

For license information, please see the file COPYING.LIB.
